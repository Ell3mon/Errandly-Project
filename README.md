## Artefactos del Proyecto

Este repositorio único gestiona los 3 componentes principales requeridos:

1.  **Base de Datos (SQL):** Contiene el esquema lógico (`Errandly_BD.sql`).
2.  **Diseño (UI/UX):** Archivos y capturas de pantalla que representan el prototipo de Figma.
3.  **Desarrollo (Code):** Carpeta lista para el código fuente (JavaScript).

## Flujo de Git

El trabajo se organiza mediante el flujo de SCRUM:

* **`main`**: Versión estable y etiquetada.
* **`develop`**: Rama de integración del trabajo del *Sprint*.
* **`feature/...`**: Ramas de trabajo para cada Historia de Usuario.
